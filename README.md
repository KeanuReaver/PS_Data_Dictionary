# UCSD Data Dictionary

An interactive PowerSchool plugin that provides a categorized data dictionary with support for user-made tables. Designed to replace the static PowerSchool table PDF with live metadata and navigation.

## 📁 Folder Structure

- `project/` – All files needed for the plugin installation
- `archive/` – Historical versions (ZIP format, dated)
- `resources/` – (Optional) Dev-only tools or references

## ⚙️ Features

- Browse all PowerSchool tables and fields
- Highlight core vs extended schema
- Supports user-created tables
- Quick search and deep linking

## 🛠️ Development Notes

This plugin is built using HTML, AngularJS, and RequireJS inside PowerSchool's plugin framework. To deploy, zip the `project/` folder and install via PowerSchool Plugin Management.
